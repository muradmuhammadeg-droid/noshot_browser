An ultra-minimalist, medium-RAM desktop browser framework that replicates the clean aesthetic of Google Chrome while implementing an ethical background ad-blocking core.

## Core Features

* **Chrome Minimalist Interface**: Pill-shaped capsule address input field, Material-style circular navigation buttons, and integrated layout tabs.
* **Medium RAM Footprint**: Bypasses Google Chrome's heavy process-per-tab resource drain by running an optimized single-window canvas layout.
* **Ethical Adblocking Core**: Automatically strips data-tracking and intrusive advertising scripts across the web.
* **Creator Whitelist Guard**: Intentionally suspends adblocking scripts whenever **Photopea** or **Aternos** are active to ensure free platform sustainability.

## How to Run Locally

To boot up the browser framework on your local machine, make sure you have [Node.js](https://nodejs.org) installed, then execute these commands in your terminal or PowerShell:

```bash
# 1. Clone the repository
git clone https://github.com/muradmuhammadeg-droid/noshot_browser.git

# 2. Enter the project directory
cd noshot_browser

# 3. Install the Electron dependencies
npm install

# 4. Fire up the browser application
npm start
```
